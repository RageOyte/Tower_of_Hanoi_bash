# Tower_of_Hanoi_bash
A terminal-based visual implementation of the classic **Tower Of Hanoi** puzzle written entirely in **Bash scripting**.
This project demonstrates:
- Recursive programming in Bash
- Dynamic array manipulation
- Iteration and recursion concepts
- Terminal-based visualization
- Simple animation using `sleep`
- Logging program output to a file
